Projects

Some of my more recent work

VR Training app for Police

Led development of a training app for a UK police force. Built for Pico Neo 3 using Unity XR libraries.

  • Some of the systems developed:
  • - Fully interactive VR environments with dynamic physics based interactions.
  • - Openable doors, drawers, cupboards, kitchen appliances (anything with a handle)
  • - Fully grabbable items with individual hand poses
  • - Dynamic scene switching using a persisted VR Rig
  • - Tagging system using Unity Pointer events for displaying object information
  • - NPC's using a trigger based dialogue system
  • - Scoring system
  • - Zone barriers for splitting up the environment
  • - Skin tone switcher for VR Hands

Fully Interactive VR Environments

Optimised for mobile VR and deployed to Oculus Quest and Pico Neo Headsets. These environments provide high levels of immersion through natural physics based interactions.

Hololens Manufactoring Demo

A Hololens demo using Mixed Reality Toolkit and Vuforia to list components and display information for selected items. Vuforia allowed for image tracking to be possible to align the box components when a carefully placed image marker is placed on the box.

Talkout VR App

Sole developer for a mobile VR application deployed to iOS and Android for potraying a stressful work environment.

  • - The app uses authentication and deeplinking from the talkout app to check a users credentials before entering.
  • - VR was achieved using the Open Source Cardboard SDK in Unity.
  • - Integrated Firebase for providing chrashlytics and monitoring user logs
